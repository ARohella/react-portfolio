# Akshat Rohella - Portfolio

![React](https://img.shields.io/badge/React-20232A?style=for-the-badge&logo=react&logoColor=61DAFB) ![TypeScript](https://img.shields.io/badge/typescript-%23007ACC.svg?style=for-the-badge&logo=typescript&logoColor=white) ![Sass](https://img.shields.io/badge/Sass-CC6699?style=for-the-badge&logo=sass&logoColor=white)

My personal portfolio website showcasing projects, experience, skills, and certifications.

**Live at [akshat-rohella.vercel.app](https://akshat-rohella.vercel.app/)**

## Tech Stack

- React 18 with TypeScript
- SCSS for styling
- Material-UI components
- React Router for navigation
- EmailJS for contact form
- Deployed on Vercel

## Local Development

1. Install dependencies:

    ```bash
    npm install
    ```

2. Start the development server:

    ```bash
    npm start
    ```

3. Open [http://localhost:3000](http://localhost:3000) to view in the browser.
